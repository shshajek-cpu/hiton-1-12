## 주요 기능

### 1. 컨텐츠 진행 체크
- 초월, 원정, 성역, 슈고페스타, 사명 등 **주간 컨텐츠**
- 일일던전, 각성전, 토벌전, 악몽, 차원침공 등 **일일 컨텐츠**
- 클릭 한 번으로 클리어 체크
- 충전권 보너스 횟수 별도 관리

### 2. 아이템 수입 관리
- 획득 아이템 등록 (등급, 수량, 가격)
- 판매중 → 판매완료 상태 변경
- 오늘/이번주/이번달 총 수입 확인

### 3. 멀티 캐릭터 지원
- 여러 캐릭터 등록
- 캐릭터별 탭으로 빠른 전환
- 총합 대시보드에서 전체 현황 확인

### 4. Google 계정 연동
- 로그인 한 번으로 여러 기기 동기화
- PC에서 기록 → 모바일에서 확인
